1Globe

The world's most multilayered globe.


Earth's imagery is displayed on a digital globe, which displays the planet's surface using different layers images from different services. After zooming in far enough, the imagery transitions into different imagery of the same area with finer detail.


Launch globe

Google Chrome and Microsoft Edge browsers are supported.

 

 

 

Features

Basic

The 1Globe is a three-dimensional (3D) software model representation of the Earth wich provides the user with the ability to freely move around in the virtual environment by changing the viewing angle and position.

1Globe geo-browser is a free, virtual globe for Web. The geo-browser provides a convenient interface for interacting with maps, imagery and geospatial data in the context of a 3D globe.

Layers

The virtual globe have the capability of representing many different view-layers on the surface of the Earth. You can change order, visibility and opacity for every layer.

Elevation

1Globe Terrain Engine fuses several data sources into a single set optimized for 3D map visualization.

Also separate using CGIAR CSI SRTM DEM. The SRTM 90m digital elevation data, produced by NASA originally, covering all of the countries of the world.

Caching

Caching component stores data so that future requests for that data can be served faster. A cache hit occurs when the requested data can be found in a cache.

Standalone

1Globe geo-browser uses PWA technologe. A progressive web application is a type of application software delivered through the web.

This mode allows to run the application locally on the device and doesn't require anything else to be functional. It's will allow you to view the globe without having an active connection to internet. It uses previously loaded CGIAR CSI SRTM elevation data and caching imagery.

Geolocation

The device's IP address use to determine the country, city, and ZIP code, determining its geographical location.

Search

The Search allows you to look up a location from a textual name, description, address or geo-coordinates.

Application uses seivices such as Bing Maps Locations, Nominatim (OpenStreetMap data) and Yandex.Search seivice. The Services supports structured and free-form search queries.

Panels

Layers: toggles the display of the layer list where you control the content displayed on the globe.

Settings: opens dialog of the settings where you configure the geo-browser.

Infopanel: information panel that displays data for the currently position on the globe.

Layers

MsBing Satellite

The Bing Maps aerial imagery web mapping service offers worldwide orthographic aerial and satellite imagery. Coverage varies by region, with the most detailed coverage in the USA and United Kingdom. Coverage in different areas within a country also varies in detail based on the availability of imagery for that region. Bing Maps is continuously adding imagery in new areas and updating coverage in areas of existing coverage.

Google Satellite

Google Maps is a free online map from Google. It's accessible through your web browser or as an app for mobile devices. You can use Google Maps to get step-by-step directions, find information about local businesses, and a whole lot more.

Yandex Satellite

Yandex Maps is a Russian web mapping service developed by Yandex. The service provides detailed maps of the whole world. It includes a search, information about traffic jams, routing and street panoramas. Yandex.Maps is the most popular cartographic service in Russia.

ArcGIS Satellite

World Imagery provides one meter or better satellite and aerial imagery in many parts of the world and lower resolution satellite imagery worldwide. In addition to commercial sources, the World Imagery map features high-resolution aerial photography contributed by the GIS User Community.

M-F-Free Relief

The vision of Maps-For-Free is to offer free worldwide relief maps and other layers which can easily be integrated into existing map projects.

NatGeo Terrain

This map portrays surface elevation as shaded relief. This map is used as a basemap layer to add shaded relief to other GIS maps, such as the ArcGIS Online World Street Map. It is especially useful in maps that do not contain orthoimagery. The shaded relief imagery was developed by Esri using GTOPO30, Shuttle Radar Topography Mission (SRTM), and National Elevation Data (NED) data from the USGS.

ArcGIS Terrain

This map portrays elevation as an artistic hillshade. This map is designed to be used as a backdrop for topographical, soil, hydro, landcover or other outdoor recreational maps. The map was compiled from a variety of sources from several data providers.

Google Terrain

Google Maps is a free online map from Google. It's accessible through your web browser or as an app for mobile devices. You can use Google Maps to get step-by-step directions, find information about local businesses, and a whole lot more.

ArcGIS Physical

This map presents the Natural Earth physical map at 1.24km per pixel for the world and 500m for the coterminous United States.

ArcGIS NatGeo

This component tile layer displays Ecological Land Units (ELU) and bathymetry fused with multi-directional hillshade and undersea relief at small to mid scales. It is designed for use as a component base layer in the National Geographic Style Map. It is not intended for stand-alone use.

NatGeo World

This map is designed to be used as a general reference map for informational and educational purposes as well as a base map by GIS professionals and other users for creating web maps and web mapping applications. The map was developed by National Geographic and Esri and reflects the distinctive National Geographic cartographic style in a multi-scale reference map of the world.

NatGeo Topo

This map is designed to be used as a basemap by GIS professionals and as a reference map by anyone. The map includes administrative boundaries, cities, water features, physiographic features, parks, landmarks, highways, roads, railways, and airports overlaid on land cover and shaded relief imagery for added context. This basemap was compiled from a variety of best available sources from several data providers, including USGS, U.S. EPA, U.S. NPS, FAO of the United Nations, Department of NRCAN, GeoBase, Agriculture and Agri-Food Canada, Garmin, HERE, Esri, OpenStreetMap contributors, and the GIS User Community.

OpenTopoMap

OpenTopoMap is a free, topographic map that is generated from the data of the OpenStreetMap and SRTM elevation data. The map style is based on the official maps and relies on good legibility through high contrast and balanced signatures.

OpenStreetMap

OpenStreetMap is a collaborative project to create a free editable map of the world. The geodata underlying the map is considered the primary output of the project. The creation and growth of OSM has been motivated by restrictions on use or availability of map data across much of the world, and the advent of inexpensive portable satellite navigation devices.
